Detchun

Objective:
The Detchun ERP WebApp was developed to meet the comprehensive management and monitoring needs of Detchun's core business operations, including quotations, delivery management, inventory, receivables, and loading processes. This solution enhances operational efficiency and provides detailed, transparent reports to support quick and accurate decision-making. The multi-language system is designed to accommodate branch employees in different countries.
Results and Benefits Achieved:
Minimized time for manual report compilation. Improved monitoring and management capabilities through visual reports and an automated alert system. Increased accuracy in tracking transactions and customer receivables. Enhanced performance in processing large data sets with advanced technology solutions.
Overall Evaluation:
The Detchun WebApp has become a powerful tool for Detchun in managing critical business processes, ensuring timely, accurate, and synchronized information. Its simple, user-friendly interface is compatible with both mobile phones and computers, with dedicated pages for each category, allowing users to easily track and retrieve detailed information. This is a significant milestone in the company's digital transformation journey, strengthening its competitive edge in the market.

Technogy used: C# ASP.NET core API ASP.NET core MVC HTML5 JavaScript CSS SQL Server T-SQL Bootstrap
